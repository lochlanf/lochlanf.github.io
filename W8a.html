13.1 Learning Objectives

At the conclusion of today’s workshop students are expected to be able to:

  • Understand the strengths and weaknesses of using bash for bioinformatics
  • Learn about the benefits of R for bioinformatics
  • Produce a working installation for R and RStudio on their personal machine

13.2 Vectors

Vectors are one-dimensional and are arguably R’s simplest type of data object.

13.3 > Exercise 1: Create a numeric vector called x that starts at 5 and goes up to 500 in increments of 1.

Exercise 2: Create a character vector called y where the first five values are “A” and the last three are “B”.

13.4 Variable types

In R, it is often essential to recognize the specific type of data stored in an object, as it affects the operations and functions that can be applied to it.

Exercise 3: Use the is.factor(), is.numeric(), is.character(), and str() commands on x and y to show their types and structure.

Exercise 4: Create a character vector with 50 values of “A”, 50 values of “B”, and 50 values of “C”. (Hint: Use the rep(X,N) command to repeat a given value X N times)

13.5 Factors

Factors in R are used to work with categorical data, which can be ordered or unordered. A factor’s levels define the possible values that observations in the data can take.

Exercise 5: Create a factor from your character vector (from Exercise 4), with levels “A”,“B”, and “C” in that order, as well as another factor where the levels “C”,“B”, and “A” in that order.

Exercise 6: Convert each of your new factors to integer vectors.

13.6 Matrices

Matrices are two-dimensional arrays that can store data of a single type. They are useful in various statistical analyses and operations.

Exercise 7: Create a numeric vector of length 9, then use that numeric vector to define a matrix with 3 rows and 3 columns.

Exercise 8: Use the ? command to look at the options for the matrix() command. Create a new 3 x 3 matrix using the byrow argument. How does this differ from your results for Exercise 7?

13.7 Data Frames and lists

Data frames and lists allow us to store and manage data more flexibly, as they can contain various types of data structures. Understanding them is key to data management and manipulation in R. Let’s create and inspect data frames and lists through the following exercises:

Exercise 9: Create a data frame with your original character vector (from Exercise 4), the two factors (from Exercise 5), and the two integer vectors (from Exercise 6). Please also include column names.

Exercise 10: Create a list containing x and y from Exercises 1 and 2, as well as your data frame from Exercise 9.
